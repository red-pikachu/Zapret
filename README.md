# Zapret macOS Native App

Нативное macOS-приложение (Status Bar) для управления [zapret2](https://github.com/bol-van/zapret2). Позволяет обходить DPI и замедления популярных ресурсов (YouTube, Discord, Telegram и др.) прямо из системного трея.

![App Icon](icon.png)

## Основные возможности

*   **Minimalist UI**: Живет в Menu Bar (рядом с часами), не занимает место в Dock.
*   **Passwordless Sudo**: Один раз настраивается доступ, после чего включение/выключение происходит мгновенно без ввода пароля.
*   **Динамические стратегии**: Поддержка JSON-конфигов стратегий, которые можно обновлять "по воздуху".
*   **Native Notifications**: Системные уведомления о статусе работы и обновлениях.
*   **Custom Configs**: Легкое редактирование `hostlist.txt` через папку `~/.zapret2`.

## Установка и запуск

1.  Установите зависимости через Homebrew:
    ```bash
    brew install pkg-config
    ```
2.  Скачайте `Zapret.app` (или соберите из исходников).
3.  Переместите приложение в папку `/Applications`.
4.  В терминале выполните команду для снятия карантина:
    ```bash
    sudo xattr -rd com.apple.quarantine /Applications/Zapret.app
    ```
5.  В меню приложения выберите **Настройки -> Разрешить запуск без пароля...** и введите пароль администратора один раз.
6.  Нажмите **Запустить Zapret**.

## Управление стратегиями

Приложение читает список стратегий из файла `~/.zapret2/strategies.json`, но при запуске автоматически проверяет обновления с GitHub.

### Синхронизация своего списка (для разработчика)
Если вы добавили новую стратегию в локальный файл `strategies.json` в корне проекта и хотите, чтобы она появилась у всех пользователей приложения:
1.  Отредактируйте `strategies.json`.
2.  Запустите скрипт синхронизации:
    ```bash
    ./sync_strategies.sh "Описание обновления"
    ```
3.  Приложение при следующем запуске само подтянет новый список из репозитория.

## Технические подробности

*   **Движок**: Использует скомпилированный под Darwin бинарник `tpws`.
*   **Перехват**: Использует системный фаервол `pf` (Packet Filter).
*   **QUIC Block**: Автоматически блокирует UDP 443 порт для форсированного перехода браузеров на TCP (необходимо для работы Ютуба).
*   **Язык**: Swift 6 (AppKit), Bash (wrapper).

## Сборка

Если вы хотите собрать приложение самостоятельно:
1.  Установите `xcodegen`: `brew install xcodegen`.
2.  Выполните `xcodegen` в корне проекта.
3.  Откройте `Zapret2Mac.xcodeproj` в Xcode и соберите (Product -> Build).

Подробная информация о тонкостях работы на macOS находится в файле [TECH_NOTES.md](TECH_NOTES.md).
